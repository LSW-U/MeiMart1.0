# 隐私政策

最后更新：2026-06-15

## 1. 我们收集什么

MeiMart（以下简称"我们"）尊重您的隐私。本应用收集以下信息：

### 1.1 您主动提供的信息

- 手机号（用于登录）
- 收货地址
- 订单信息
- 反馈内容

### 1.2 自动收集的信息

- 设备型号、操作系统版本
- 网络连接状态
- 应用使用统计（崩溃报告）

### 1.3 权限使用

| 权限     | 用途                    |
| -------- | ----------------------- |
| 相机     | 上传商品/评价图片、扫码 |
| 相册     | 选择图片上传            |
| 位置     | 地址自动填充            |
| 网络     | 数据同步                |
| 推送通知 | 订单/物流提醒           |

## 2. 我们如何使用

- 提供商品展示、下单、配送服务
- 改进应用性能和稳定性（崩溃报告）
- 防止欺诈和滥用

## 3. 数据存储

- 敏感凭证（token）存储于设备 SecureStore（iOS Keychain / Android Keystore）
- 购物车等数据存储于本地 AsyncStorage
- 服务器数据通过 HTTPS 加密传输

## 4. 第三方服务

| 服务         | 用途                       |
| ------------ | -------------------------- |
| Sentry       | 错误监控（不发送个人信息） |
| Expo Updates | OTA 更新                   |

## 5. 您的权利

- 访问、修改、删除您的个人数据
- 撤回权限授权
- 注销账户
